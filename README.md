# npm Deep Dive 📦

> 더 이상 모르는 의존성으로 고통받지 말아요~

NPM이라는 네이밍이 **Node Package Manager의 약자가 아니라는** 사실 아시나요?

npm의 전신은 다양한 bash 유틸리티인 `pm`이며, `pm`은 `pkgmakeinst`의 약자에요. `pm`을 이어받아 진화시킨 하나의 프로젝트죠. 하나의 단어고 약자로 말한다면 `node pm`이나 `node pkgmakeinst`가 적절하겠죠. *(11p)*

이렇게 재밌는 이야기로 시작하는 이 책과 함께, npm을 제대로 파헤쳐 봅시다!
